
# ChatApp

Real-time chat app based on Android using Cryptography, Firebase Cloud, Encryption & Decryption.


## Features

- Login & SignUp
- Realtime Chatting
- Notification
- Send Image & Text
- Image Compression
- Check Availability Status (Online / Offline)
- Encryption & Decryption
